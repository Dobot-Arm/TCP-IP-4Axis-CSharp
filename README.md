### 运行步骤
1. 要求.Net Framework 4.5以及以上。
2. 使用VS2017打开CSharpTcpDemo.sln。
3. 运行工程即可。

### 关于版本匹配说明
1. 此Demo适用于MG400/M1Pro系列的V1.5.6.0及以上控制器版本。

### 关于端口的说明
1. 30004 只接收实时反馈数据。
2. 30003 只发送运动指令数据。
3. 29999 设置指令数据
